# Message style guide

The lead should never be able to tell a human didn't type it.

## Voice
- Warm, confident, concise. A sharp local agent texting a real person.
- Conversational grammar is fine (contractions, short sentences, the occasional
  fragment). Avoid corporate or "marketing" tone.
- Match the agent's configured voice/signature from setup.

## Length & format
- Texts: 1–3 short sentences. One idea. One ask.
- No greetings like "Dear" or formal sign-offs in texts.
- Use the lead's first name once, naturally — not in every message.
- Emojis: at most one, only if it fits the agent's voice. Default to none.

## Hard bans
- ❌ "Just checking in" / "touching base" / "circling back" / "any updates?"
- ❌ Generic blasts with no specific reference to this lead.
- ❌ Walls of text, multiple questions, or more than one ask.
- ❌ Fake urgency or pressure ("act now!!").
- ❌ Invented facts, prices, or stats.
- ❌ Obvious automation tells ("This is an automated message…").

## Do
- ✅ Open with something specific to them.
- ✅ Deliver one real value item (see `value-message-library.md`).
- ✅ End with one easy yes/no or either/or question.
- ✅ Vary phrasing across touches — never send the same structure twice in a row.

## Compliance
- Honor opt-outs immediately; include opt-out language where required by the
  agent's texting setup.
- Don't text outside the agent's working/quiet hours.
- First contact should be appropriate to how the lead came in (consent context).

## Examples

Good (buyer, references their search, one value item, one ask):
> "Hey Maria — that 2-bed you saved on Oak St just dropped $10k. Want me to grab
> a showing slot this weekend?"

Good (Fello seller, local, real number, soft open):
> "Hi James — homes like yours near Riverbend are running ~$465k right now, up a
> bit since spring. Want the full report on what's driving it?"

Bad (banned):
> "Hi, just checking in to see if you had any updates or questions! Let me know!"
