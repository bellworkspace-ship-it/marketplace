# Cadence rules

Match follow-up timing to lead temperature. The cadence adapts — these are
starting points the learning log will tune over time.

## New lead (just came in)
**Speed is everything.** First touch within minutes of detection, regardless of
hour (respecting the agent's quiet hours for sending). If no reply:
- Touch 2: same day, a few hours later, different value angle.
- Touch 3: next day.
- Touch 4: day 3.
- Touches 5–8: spaced every 2–3 days across the first 2 weeks.
- Then move to warm/long-term nurture if still no engagement.

Each touch uses a **different value type** — never repeat the same angle back to
back.

## Warm lead (engaged, not committed)
- Every 4–7 days, always with new value.
- Tighten to every 2–3 days if they re-engage; loosen if they go quiet.
- Always have a logged next-touch task with a due date.

## Hot lead (asking about specific homes / timing)
- Respond within minutes, every time.
- Drive toward a booked appointment; don't let it cool.

## Long-term nurture (cold after full cadence, or "months out")
- Roughly every 2–4 weeks with genuinely useful value (market updates, equity
  changes, notable listings).
- Re-promote to warm the moment they reply or show new property activity.

## Adaptive rules
- **Responsiveness raises frequency; silence lowers it** (and switches the value
  angle and time of day).
- **Property activity = buying signal** — a lead viewing listings jumps the queue.
- **Fello score spike** (intent/equity change synced into FUB) → re-engage now.
- Never exceed one automated message per lead per run cycle.
- Stop the moment a lead opts out or asks to pause.

## Timing
- Respect the agent's working/quiet hours from setup.
- Bias sends toward windows the learning log shows get replies for this team
  (often evenings and weekends, when most leads are actually free).
