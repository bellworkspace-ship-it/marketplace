# Reply interpretation & objection playbook

First classify the reply, then respond. Always add value; never argue.

## Step 1 — classify the reply
- **Hot** — asks about a specific property, price, showing, or timing → move to
  book an appointment now.
- **Warm** — engages but non-committal ("still looking," "maybe in a few
  months") → keep nurturing with value, set the next cadence step.
- **Objection** — a reason not to engage (see below).
- **Cold/none** — no reply → continue cadence with a new value angle.
- **Out-of-area home-value** → referral flow (`referral-routing-revii.md`).
- **Stop/opt-out** → mark do-not-contact, honor immediately, log it.
- **Sensitive** (complaint, legal, distress, major life event) → hold for the
  agent, do not auto-respond.

## Step 2 — handle the common objections
Acknowledge → add value → soft next step. One message, not a lecture.

**"I already have an agent."**
> "Totally respect that — sounds like you're in good hands. If you ever want a
> quick second opinion on value or a specific listing, I'm here. Want me to keep
> sending {area} homes that match in the meantime?"

**"Just looking / not ready."**
> "No rush at all. Want me to set a live alert so you only hear from me when
> something that actually fits {criteria} pops up? Keeps you in the loop with
> zero pressure."

**"Too expensive / market's crazy."**
> "Fair — and timing matters. Inventory in {area} is up {n}%, which is giving
> buyers more negotiating room than they had. Want me to show you a couple priced
> with room to move?"

**"Just wanted my home value" (local seller).**
> "Here's the number: ~{value}, up about {pct} recently. Biggest driver is
> {factor}. Want the full report, or are you weighing a possible move?"

**"How much is your commission / what do you charge?"**
> Answer honestly and briefly with the agent's real terms, then redirect to value
> and a call. Never improvise numbers — pull the agent's actual terms or hold.

**Silence after value touches.**
> Switch value type (listing → market stat → equity update), lengthen the gap,
> and try a different time of day. After the cadence is exhausted, move to
> long-term nurture and tell the agent.

## Rules
- One acknowledgment, one value item, one ask. No pressure, no guilt.
- Never dispute the lead's stance — reframe with something useful.
- Pull real numbers; if you don't have them, ask or hold rather than guess.
- Log the objection type in the learning log so the team learns what rebuttals
  convert.
